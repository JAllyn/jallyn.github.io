# Riverlands Rugby Football Club - Static Site

**Fear the Boars**

This is a fully static, modern HTML/CSS/JS version of the original WordPress site at riverlandsrugby.com.

## What's Included

- **index.html** — Beautiful homepage with hero image, quick links, and calls to action
- **history.html** — Club history and championship timeline
- **practices.html** — Current practice schedule and location info
- **rugby-101.html** — Introduction to the game for new players
- **events.html** — Events calendar and Battle of the Burg info
- **officers.html** — Meet the club leadership
- **join.html** — How to get involved / contact the coaches

## Design

- Modern, fast, mobile-first responsive design
- Dark theme with emerald green and gold accents (matching "Golden Boars")
- Custom logo and hero imagery generated specifically for the club
- Uses Tailwind CSS via CDN + Font Awesome icons
- No external dependencies or tracking

## How to Use

1. Download or copy the entire `riverlands-rugby-static` folder
2. Open `index.html` in any browser
3. For production, upload the folder to any static hosting (Netlify, Vercel, GitHub Pages, Cloudflare Pages, etc.)

## Notes on Conversion

The original site (riverlandsrugby.com) is a WordPress site with limited public text content visible to crawlers. This static version faithfully recreates and expands upon all publicly available information from the site and associated social channels (Instagram @rrfc_golden_boars, Facebook) while providing a much faster, cleaner experience.

Key content sources:
- Club name, tagline, and "Fear the Boars" branding
- Officer names and roles
- Practice location (Falmouth, VA)
- Championship history (2022 CRU & MAC South champions + recent D2 titles)
- Events (AGM, 7s practices, Battle of the Burg)

## Customization

All files are plain HTML. Edit any `.html` file directly. The Tailwind classes and structure are easy to modify.
